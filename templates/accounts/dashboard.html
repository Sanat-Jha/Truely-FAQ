Integration Guide

To integrate TrulyFAQ with your website, follow these steps:

  1. Add a website from the dashboard
  2. Get your API key from the website detail page
  3. Use our API endpoints to submit questions and retrieve FAQs

Example: Submit a question

fetch('https://truelyfaq.com/api/questions/submit/', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
        'X-API-Key': 'your-api-key'
    },
    body: JSON.stringify({
        user_email: 'user@example.com',
        question_text: 'How do I use this service?'
    })
})
