TrulyFAQ Documentation

Learn how to integrate TrulyFAQ into your website.

Getting Started

  1. Create an account: Sign up for a TrulyFAQ account.
  2. Add your website: From your dashboard, add the website where you want to implement TrulyFAQ.
  3. Get your API key: Each website has a unique API key that you'll need for integration.

API Integration

TrulyFAQ provides two simple API endpoints for submitting questions and retrieving FAQs.

Submit a Question

Use this code to submit a question from your contact form:

const response = await fetch(`{hosting url}/api/questions/submit/`, {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
        'X-Api-Key': this.apiKey
    },
    body: JSON.stringify({
        user_email: email,
        question_text: question
    })
});

Fetch FAQs

Use this code to retrieve FAQs for display on your website:

const response = await fetch(`{hosting url}/api/faqs/`, {
    method: 'GET',
    headers: {
        'X-Api-Key': this.apiKey
    }
});

The response will contain a JSON object with FAQs that you can display on your site.

Dashboard Usage

After logging in, you can:

  • Manage Websites: Add, view, and delete websites from your account.
  • Answer Questions: View and respond to user questions.
  • Manage FAQs: Edit, delete, or toggle visibility of automatically generated FAQs.
