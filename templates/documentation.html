TrulyFAQ Documentation

Learn how to integrate TrulyFAQ into your website.

Getting Started

  1. Create an account: Sign up for a TrulyFAQ account.
  2. Add your website: From your dashboard, add the website where you want to implement TrulyFAQ.
  3. Get your API key: Each website has a unique API key that you'll need for integration.

API Integration

TrulyFAQ provides two simple API endpoints for submitting questions and retrieving FAQs.

Submit a Question

Use this code to submit a question from your contact form:

const response = await fetch(`https://truelyfaq.pythonanywhere.com/api/questions/submit/`, {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
        'X-Api-Key': this.apiKey
    },
    body: JSON.stringify({
        user_email: email,
        question_text: question
    })
});

Fetch FAQs

Use this code to retrieve FAQs for display on your website:

const response = await fetch(`https://truelyfaq.pythonanywhere.com/api/faqs/`, {
    method: 'GET',
    headers: {
        'X-Api-Key': this.apiKey
    }
});

The response will contain a JSON object with FAQs that you can display on your site.

Dashboard Usage

After logging in, you can:

  • Manage Websites: Add, view, and delete websites from your account.
  • Answer Questions: View and respond to user questions.
  • Manage FAQs: Edit, delete, or toggle visibility of automatically generated FAQs.

Answering User Questions

When users submit questions through your website, they appear in your dashboard for you to answer:

  1. Navigate to the specific website by clicking "Manage Website" from your dashboard
  2. You'll see a list of all questions, with unanswered questions highlighted
  3. Click on any unanswered question to open the answer form
  4. Type your response in the answer field and click "Submit Answer"
  5. The system will automatically:
    • Mark the question as answered
    • Send an email notification to the user who asked the question
    • Generate a new FAQ entry based on the question and your answer

Managing FAQ Visibility

TrulyFAQ automatically generates FAQ entries from answered questions. You can control which FAQs are visible on your website:

  1. Navigate to the "FAQs" section of your website's detail page
  2. Each FAQ has a visibility toggle button (Show/Hide)
  3. Click the toggle button to instantly change the visibility status:
    • Visible FAQs will appear when users fetch FAQs through the API
    • Hidden FAQs will be stored in your account but won't be publicly accessible
  4. The visibility status is indicated by a color-coded badge (green for visible, gray for hidden)
  5. Changes take effect immediately - no need to save or refresh

This visibility toggle feature allows you to:

  • Curate which FAQs are most relevant to your users
  • Temporarily hide FAQs that need revision
  • Control the number of FAQs displayed on your website
  • Hide duplicate or similar questions while keeping the information in your database
